
# YoutubeMusicClone
![yt_music_full_rgb_black](https://github.com/yahoth/YoutubeMusicClone/assets/99034396/dff897a5-f5b5-4bf7-859f-7a9c2460ae57)

# **기술 스택:**

**구현환경**: iOS 14.3

**프로그래밍 언어:** Swift 5.8

**프레임워크:** UIKit, SwiftUI, Combine, AVFoundation, KingFisher

**도구:** Xcode, Git, Figma(UI분석)

**기술:** RESTful API, Storyboard, MVVM

# 핵심 **기능**

- 컬렉션 뷰를 사용한 섹션별 음악 레이아웃 구성
- Spotify API를 활용한 데이터 로드 및 처리
- 음악 감상 및 플레이어 조작 기능
- 실시간 검색 기능

# 구동방법
1. 이 프로젝트를 설치하세요
2. [Spotify 개발자용 웹사이트](https://developer.spotify.com) 에서 Spotify 계정을 생성/로그인하세요.
3. DashBoard로 가서 Create app을 선택하세요
4. 원하는 이름과 설명을 입력하고 대시보드에서 방금 만든 앱을 선택하세요
5. Setting을 선택하고 Basic Information에서 ClientID와 ClientSecret을 복사합니다
6. 프로젝트에 있는 **UserSensitiveData.swift** (./YoutubeMusicAppClone/UserSensitiveData)을 엽니다.
7. 해당 파일에 있는 변수에 값을 할당하면 준비가 완료됩니다
